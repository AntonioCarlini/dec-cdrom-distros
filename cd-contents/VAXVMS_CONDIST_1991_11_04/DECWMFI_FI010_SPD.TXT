          DECwindows-työpöytäohjelmisto on VMS-käyttöjärjestelmän graa-
          finen käyttöliittymä. DECwindows-sovellusten pohjana käytetään
          Open Software Foundationin Motif-käyttöliittymää. OSF/Motif on
          graafinen käyttöliittymä, joka on saavuttanut teollisuusstan-
          dardin aseman. Käyttöliittymän ansiosta käyttäjä voi toteuttaa
          VMS-käyttöjärjestelmän toimintoja tehokkaalla osoitus ja pai-
          nallus -menetelmällä. DECwindowsiin kuuluu joukko integroituja
          työpöytäsovelluksia, joiden käyttö perustuu edellä mainittuun
          periaatteeseen. Nämä sovellukset muodostavat VMS-käyttäjän pe-
          rustyökalut.

          VMS DECwindows Motif -käyttöliittymä perustuu MIT:n X Win-
          dow -järjestelmän määritykseen (versio 11, release 4). Oh-
          jelmistoon kuuluvat seuraavat OSF/Motif V1.1.1.[*] X Window
          -järjestelmästandardit: X11-verkkoprotokolla, perustyöasema-
          kirjasimet, C-kielinen yhteys Xlib-ohjelmointikirjastoon ja
          C-kielinen yhteys Xtoolkit-kirjastoon. Työasemakäyttäjiä varten
          DECwindowsiin kuuluu X11-protokollaan integroitu Adobe Display
          PostScript -järjestelmän tuki.

          X11-verkkoprotokollan tuki asiakaskirjastossa ja näyttöpalve-
          limessa antavat VMS-käyttöjärjestelmälle mahdollisuuden toimia
          hajautetusti yhdessä muiden X11-yhteensopivien järjestelmien
          kanssa.

          [*] OSF:n toimittama ohjelmisto ei ehkä ole yhteensopiva DECwin-
          dowsin tulevien versioiden kanssa.

          Lisenssin haltija sitoutuu käyttämään Display PostScript -
          ohjelmaa vain niissä Digitalin laitteistoissa, jotka tässä
          ohjelmistotuotekuvauksessa on määritetty lisensioiduiksi lait-
          teistoiksi. Lisäksi lisenssin haltija sitoutuu olemaan käyttä-
          mättä ohjelmistoa, joko suoraan tai epäsuorasti, tulostaakseen

          DECwindows-ohjelmisto tukee X Window System -järjestelmän
          asiakas-palvelin-periaatetta. Tämän periaatteen avulla X Window
          -ohjelmia voidaan toteuttaa yhdessä koneessa ja niiden tulos
          voidaan näyttää jossain toisessa koneessa käyttämällä jotain
          tietojen siirtokäytäntöä koneiden välillä. DECwindowsiin kuu-
          luu kolme siirtoliittymää, paikallinen yhteismuisti, DECnet
          ja TCP/IP (käytettäessä Digitalin VMS/ULTRIX Connection (UCX)
          -ohjelmistoa). Käyttäjät voivat myös käyttää muita siirtoalijär-
          jestelmiä ja siirtoliittymiä.

          Lisätietoja siitä, voiko erikseen tilattava DECwindows- tai X-
          sovellus toimia VMS DECwindows Motif -järjestelmän alaisuudessa
          tai voiko sovellus olla yhteydessä DECwindows-ohjelmistoon, saa
          sovelluksen ohjelmistokuvauksesta.

          VMS DECwindows Motif -käyttöliittymäohjelmisto koostuu neljästä
          perusosasta:

          1. Istunnon hallinta toimii ylimmän tason käyttöliittymänä
             DECwindows-työasemaan. Sen avulla voidaan aloittaa muita
             sovelluksia, muuttaa ja hallita koko istuntoa ja määrittää
             tietoturva-asetuksia. Istunnon hallinnan avulla voidaan myös
             määrittää käyttöliittymän kieli tuetuille DECwindows-sovel-
             luksille, jos järjestelmään on asennettu vastaavankielinen
             VMS DECwindows Motif -käyttöliittymäohjelmisto ja sille on
             asianmukainen lisenssi. Lisätietoja tuetuista muunkielisistä
             käyttöliittymistä on tämän ohjelmistotuotekuvauksen koh-
             dassa Käyttöliittymän kielen tuki. Lisätietoja käännetyistä
             sovelluksista ja laitteisto- ja ohjelmistotuesta muunkieli-
             sissä käyttöliittymissä saa vastaavankielisen käyttöliittymän
             Järjestelmätuki-käsikirjasta.

          2. Ikkunan hallinnan avulla käyttäjä voi hallita näytössä

             OSF/Motif Ikkunan hallinta. Käytettävissä on myös DECwindows
             X User Interface (XUI) Ikkunan hallinta. Tällöin käyttäjä voi

             den DECwindows-sovellusten välillä. Useat ohjelmat voivat
             käyttää myös näitä ominaisuuksia ilman muutoksia.

          DECwindows-käyttöliittymän mukana toimitetaan joukko integ-
          roituja työpöytäsovelluksia. Nämä sovellukset hyödyntävät

          monenlaista hyötyä.

          Seuraavat sovellukset on muutettu OSF/Motif-käyttöliittymää
          varten ja ne kuuluvat VMS DECwindows Motif -ohjelmiston tähän
          versioon:

          o  Kirjasto - järjestelmään asennettujen käsikirjojen näyttöso-
             vellus.

          o  Laskin - tieteellinen peruslaskin.

          o  Kalenteri - henkilökohtainen ajanhallintajärjestelmä.

          o  Näytön tulostus - sovellus koko näytön tai näytön osan tie-

          PostScript -toiminto on käytettävissä CDA-asiakirjojen näyttö-
          ja Sähköposti-sovelluksessa.

          Ohjelmointituki

          VMS DECwindows Motif -ympäristöön kuuluu laaja valikoima oh-
          jelmointikirjastoja ja -työkaluja, joita voidaan käyttää uusien
          sovellusten kehityksessä.

          Ohjelmistoon kuuluvat kirjastot voidaan jakaa neljään luokkaan:
          X Window -järjestelmän vakiokirjastot, kuten Xlib ja Intrinsics,
          vanhojen XUI-sovellusten käyttöä varten tarvittavat kirjas-
          tot, OSF/Motif-käyttöliittymän kirjastot, joiden avulla voi-
          daan kehittää Motif-käyttöliittymää käyttäviä sovelluksia sekä
          Digitalin laajennuskirjastot, joiden avulla käyttäjät saavat
          lisäominaisuuksia.

          VMS DECwindows Motif -ohjelmistoon kuuluu seuraavat kahdeksan
          kirjastoa:

          o  X Window System -ohjelmointikirjaston (Xlib) tuki.

          o  X Window System Intrinsics -tuki (Xtoolkit).

          o  OSF/Motif User Interface -tuki.

          o  Digitalin laajennukset OSF/Motif-kirjastoon.

          o  X User Interface (XUI) -tuki.

          o  Työkalut, joiden avulla XUI-sovellus voidaan muuttaa
             OSF/Motif-sovellukseksi.

          o  CDA-asiakirjojen (Compound Document Architecture) tuki.

          o  DPS-tuki (Display PostScript).

          OSF/Motif-käyttöliittymän ohjelmointituki

          OSF/Motif on Digitalin ja muiden ohjelmistotoimittajien käyt-
          tämä käyttöliittymä, joka on mukautettu DECwindows-ympäristöä
          varten. OSF/Motif määrittää DECwindows-työasemassa käytettävien
          sovellusten tyylin ja käytännöt. Sovellukset käyttävät Motifin
          osia luodessaan käyttöliittymiä, jotka näyttävät ja toimivat
          samalla tavoin kuin muut Digital-tietojenkäsittely-ympäristön
          osat. OSF/Motifiin kuuluu myös tehokkaampia käyttöliittymän
          osia, gadgetteja.

          OSF/Motif Style Guide -käsikirjassa, joka kuuluu VMS DECwindows
          Motif Programming Kit -käsikirjasarjaan, kuvataan yhtenäisten
          ja hyvin integroitujen OSF/Motif-sovellusten kirjoittamisessa
          käytettävät periaatteet, käytännöt ja komponentit. Lisäksi
          VMS DECwindows Motifiin kuuluu DECwindows Companion to the
          OSF/Motif Style Guide -käsikirja. Tässä kirjassa on lisätie-
          toja OSF/Motif-käyttöliittymissä käytettävästä tyylistä. Lisäksi
          käsikirjassa on tietoja Digitalin laajennusominaisuuksista,
          jotka kuuluvat VMS DECwindows Motif -ohjelmistoon, ja luet-
          telo Digitalin suosittelemista termeistä useilla eri kielillä
          OSF/Motif-sovellusten kehittämistä varten.

          OSF/Motif-tyyliseikat on määritetty OSF/Motif Toolkit -
          kirjastossa. OSF/Motif Toolkit käyttää MIT Xtoolkit -kirjastoa,
          johon kuuluu kolme osaa:

          o  OSF/Motif-rutiinit tavallisimpia käyttöliittymän osia, kuten
             vieritysjanoja, valikoita ja painikkeita varten.

          o  Apurutiineja, joiden avulla sovellukset voivat toteuttaa
             vakiotoimintoja, kuten leikkaus ja lisäys.

          o  Resurssien hallintarutiineja tehtävää käyttöliittymän mää-
             ritystiedostojen latausta ja widget- ja gadget-ohjelmien
             luomista varten määritystiedoston tietojen perusteella.

          OSF/Motif Toolkit -kirjastoa käytetään yhdessä

          Digitalin laajennukset OSF/Motif-kirjastoon

          Digital on tehnyt laajennuksia vakio-OSF/Motif-kirjastoon.
          Näihin toimitettaviin laajennuksiin kuuluu joukko Digita-
          lin kehittämiä ohjelmia ja ajon aikaisia rutiineja, joita ei
          ole saatavissa OSF:stä. Näiden laajennusrutiinien avulla muo-
          dostetaan joidenkin DECwindowsin yleistoimintojen yhtenäiset
          käyttöliittymät. Näitä toimintoja ovat:

          o  Värin sekoitus -widget, jonka avulla käyttäjät voivat valita
             ja sekoittaa värejä käyttämällä erilaisia värimalleja.

          o  Merkkiyhdelmäwidget, johon kuuluu kansainvälisten oikealta-
             vasemmalle -merkistöjen sekä vasemmalta-oikealle-kirjoituskäytäntöjen
             tuki.

          o  Opastus-widget, jonka avulla kaikkien sovellusten opastusnäy-
             töt saadaan kuvaruutuun ja näkymään yhtenäisellä tavalla.

          o  Tulostus-widget, jonka avulla kaikki sovellukset tulostavat
             tietoja yhtenäisellä tavalla.

          o  SVN-widget (Structured Visual Navigation), jonka avulla

          VMS DECwindows Motif -ohjelmistoon kuuluu myös XUI Toolkit
          -ohjelma. XUI Toolkit ohjelma perustuu MIT R4 Xtoolkit -
          määrityksiin. Tässä ohjelmassa käytetään XUI-käyttöliittymämallia,
          johon VMS DECwindowsin aiemmat versiot perustuvat. Uudet sovel-
          lukset on kuitenkin syytä kirjoittaa OSF/Motif-käyttöliittymän
          mukaisesti, koska Motif on useiden ohjelmatoimittajien käyttämä
          vakiokäyttöliittymä.

          Ohjelmien muuntotyökalut

          Digital toimittaa ohjelmiston mukana työkalut, joiden avulla
          ohjelmoijat voivat muuntaa XUI-sovelluksen Motif-käyttöliittymää
          varten. Nämä työkalut korvaavat lähdetiedostossa olevat XUI-
          nimet vastaavilla Motif-nimillä.

          CDA-ohjelmointituki (Compound Document Architecture)

          CDA Toolkit -ohjelmaan kuuluu rutiineja, joiden avulla sovelluk-
          set voivat luoda, lukea ja tallettaa tiedostoja, jotka sisäl-
          tävät tekstiä, grafiikkaa ja kuvatietoja (yhtenäisasiakirjat).
          Näiden tiedostojen avulla tiedot voidaan tallettaa levyyn ja
          niitä voidaan siirtää sovelluksesta toiseen. Lisäksi tiedos-
          toista voidaan tulostusta varten muodostaa suuren erotuskyvyn
          grafiikkatietoja.

          Enhanced X Windows Display PostScript (XDPS)

          VMS DECwindows Motif -ohjelmistoon kuuluu Display PostScript
          -järjestelmä, joka laajentaa tavallista X-graafista ohjel-
          mointiympäristöä. Tämä laajennus on käytössä, jos käytetään
          XDPS-laajennusta tukevia työasemia.

          X Display PostScript -järjestelmä lisää seuraavat ominaisuudet
          X11-vakioympäristöön:

          o  Kaikki DECwindows-kirjasimet voidaan näyttää minkä tahansa
             kokoisina ja missä tahansa kiertokulmassa.

          o  XDPS-grafiikka määritetään käyttäjän määrittämässä koordinaa-
             tistossa, joka ei riipu käytettävän näytön erotuskyvystä.

          o  Värien tai harmaasävyjen tulkintaa muokataan automaattisesti
             näytön tyypin mukaisesti joko suoranäytön, värien rasteroin-
             nin tai puolisävytyksen avulla.

          o  DPS Display -rutiinit voidaan ladata palvelimeen ja toteuttaa
             komentojen avulla.

          o  Näytössä voidaan näyttää monipuolisia graafisia elementtejä,
             kuten tarkasti määritettyjä Bezier-käyriä.

          o  Kaikkien näyttöjen kokoa voidaan muuttaa ja niitä voidaan
             kiertää mielivaltaisesti.

          XDPS-ohjelmointi tapahtuu Abode Client Library -kirjaston väli-
          tyksellä. Lisäksi käyttäjät voivat muuntaa PostScript-tiedostoja
          C-kielisiksi ohjelmiksi pswrap-muuntimen avulla. Tämän jäl-
          keen näitä C-ohjelmia voidaan kutsua muilla ohjelmointikielillä
          tehdyistä ohjelmista.

          Display Server Extension -tuki

          VMS DECwindows Motif Display Server tukee seuraavia laajennuk-

          o  Display PostScript: Adobe Systems, Icorporatedin ja Digitalin
