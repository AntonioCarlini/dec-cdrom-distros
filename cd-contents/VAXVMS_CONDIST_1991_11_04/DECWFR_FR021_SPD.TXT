          NOM DU PRODUIT : VMS DECwindows/Français, Version 2.1  SPD 25.Z3.01

          DESCRIPTION

          VMS DECwindows/Français est l'interface française du produit
          VMS DECwindows. VMS DECwindows est inclus dans le système
          d'exploitation VMS. Pour pouvoir utiliser VMS DECwindows/Français,
          le système d'exploitation VMS et VMS DECwindows sont nécessaires.

          Le produit VMS DECwindows/Français fournit une interface en
          langue française aux applications qui créent l'environnement
          DECwindows. Cet environnement se compose de quatre composants de
          base :

          o  Le Gestionnaire de session représente le point d'entrée de
             la station de travail DECwindows. Il permet de lancer des
             applications, de personnaliser l'environnement général de la
             session, de copier et d'imprimer tout ou partie de l'écran,
             de gérer la sécurité et de contrôler la session d'une manière
             globale.

          o  Le Gestionnaire de fenêtres permet de paramétrer les
             fenêtres.

          o  Le Tableau de bord fournit une interface graphique à la
             gestion de fichiers VMS. Il permet de se déplacer dans
             l'arborescence VMS et d'effectuer des opérations sur les
             fichiers (copie, destruction, impression...).

          o  L'émulateur de terminal DECterm permet aux utilisateurs des
             stations de travail DECwindows de retrouver un environnement
             identique à celui proposé par les terminaux à matrices de
             caractères de type VT.

          DECwindows comprend les applications de bureau suivantes :

          o  Agenda - Un utilitaire de gestion du temps.

          o  Bloc-notes - Éditeur de texte.

          o  Calculatrice - Une calculatrice scientifique.

          o  Courrier - L'équivalent DECwindows de l'utilitaire MAIL de VMS.

          o  Dessin - Éditeur graphique en mode point.

          o  Émulivre - Un outil de consultation de documentation à l'écran.

          o  Horloge - Affichage de l'heure en mode numérique ou analogique
             et fonction d'alarme.

          o  Pousse-pousse - Un jeu de logique.

          o  Répertoire - Une application de stockage hiérarchique
             d'informations, reprenant les fonctionnalités d'un annuaire.

          o  Visionneuse CDA - Outil permettant d'afficher des documents
             composés (texte et graphiques conformes à l'architecture CDA)
             sur un terminal ou une station DECwindows.

          Les utilisateurs de DECwindows sur stations de travail peuvent
          afficher les fichiers au format PostScript[R] grâce à la fonc-
          tion Display PostScript[R], disponible dans les applications
          Visionneuse CDA et Courrier.

          Pour plus de détails sur VMS DECwindows, veuillez vous reporter
          à la Description de Produit Logiciel du système d'exploitation
          VMS (SPD 25.01.XX).
