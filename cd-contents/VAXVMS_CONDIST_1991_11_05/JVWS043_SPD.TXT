                                      ソフトウェア仕様書

                            仕様書の包含する範囲

     この文書は, 日本語VMS ワークステーション・ソフトウェア（以下，
     日本語VWS と呼ぶ）のソフトウェア機能仕様を述べたものです。こ
     こでは英語版VMS ワークステーション・ソフトウェアに追加, 拡張
     された日本語処理の機能仕様についてのみ記述していますので, 英語
     版に含まれている基本機能については,VMS  Workstation  Software
     の「Software  Product  Description  (SPD 28.06.11)」を参照
     してください。


1   日本語VWS の概要

日本語VWS はVAXstation ファミリー上で動作し，ビットマップ・ディスプ
レイに対する汎用的なグラフィックスおよびウィンドウィング・インターフェ
イスをユーザに提供します。

特に，日本語機能として以下が提供されます。

 ・  VT28x端末エミュレータのサポート

 ・  日本語ユーザ・インターフェイス・ルーチン
     （英語版 VWS  に以下の日本語機能が追加）

        - 漢字文字列の物理的な大きさを求める機能

        - 漢字文字列の表示機能

        - 漢字文字列の表示領域の大きさを求める機能

        - インストール検証機能


日本語VWS はVMS オペレーティング・システムのレイヤード・プロダクトで
あり，次に示すコンポーネントから構成されます。

 ・  日本語VWS

 ・  VT28xターミナル・エミュレータ[1]

 ・  日本語VWS ドキュメント


2   日本語VWS からDECwindows への移行ツール

2.1   日本語VAXuisx  ランタイム・ライブラリ/VMS  バージョン 1.0

日本語VAXuisx  ランタイム・ライブラリ/VMS  バージョン 1.0 を使用して，
日本語VWS で作成されたアプリケーションの大部分を DECwindows  環境で実
行できます。日本語VAXuisx  は共用イメージ JUISXSHR  を提供することに
より，日本語VWS のアプリケーションを再リンクしないで実行できるように
します。

特に，日本語機能として以下が提供されます。

 ・  日本語ユーザ・インターフェイス・ルーチンのサポート

 ・  日本語DECwindows  の漢字端末エミュレータの作成

2.2   UIS  Source  Code  Annotator

UIS  Source  Code  Annotator  は，日本語VWS  で作成したアプリケーショ
ンのソース・コードを解析して DECwindows  用のアプリケーションに変換す
るための情報を提供します。

2.3   制限事項

DECwindows/X11  Server  for  VMS  は，日本語環境ではサポートされませ
ん。

日本DEC のライセンス条件とその方針についての詳細は, 最寄りの日本DEC

7   ライセンス管理機能

本ソフトウェアはVMS ライセンス管理機能(LMF) をサポートしています。ラ
イセンス単位は,CPUの性能に応じて割り当てられます。

ライセンス管理機能についての詳細は,VMS Operating  System  の「Software
Product  Description(SPD  25.01.xx)」またはVMS  Operating  System の
ドキュメント・セットの中の「License  Management  Utility  Manual」を
参照してください。


8   ソフトウェア製品サービス

日本DEC では，様々なサービス・オプションを提供しています。詳細について
は, 最寄りの日本DEC 各支店/ 営業所にお問い合わせください。


9   保証

本ソフトウェアについては, 日本DEC 所定のソフトウェア保証基準に定められ
た保証が提供されます。
