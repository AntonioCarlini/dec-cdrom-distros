 PRODUCT NAME:  VAXft System Services, Version 2.0      SPD 31.64.04

 DESCRIPTION

 VAXft System Services is a OpenVMS layered software product. When com-
 bined with base OpenVMS and the VAXft series hardware, VAXft System
 Services provides the full fault tolerant system capabilities. The re-
 sulting features include the following:

 o  Automatic failover

 o  Configuration management

 o  Automatic dial-out notification (on certain system models)

 o  Component fault isolation and recovery

 o  Optional support for the DSF32-AA/AB (on certain system models)

 VAXft System Services software is used by System Managers, Operations,
 and Digital Service personnel to perform system management and main-
 tenance operations. The software is transparent to the system users
 as well as to OpenVMS layered products and applications.

 HARDWARE REQUIREMENTS

 31.64.03-x).
