
Description de Produit Logiciel


NOM DU PRODUIT: VMS DECWINDOWS/Français/CH, Version 2.1	 	SPD 25.Z9.01

DESCRIPTION

VMS DECwindows/Français/CH est l'interface française du produit VMS DECwindows.
VMS DECwindows est inclus dans le système d'exploitation VMS. Pour pouvoir
utiliser VMS DECwindows/Français/CH, le système d'exploitation VMS
et VMS DECwindows sont nécessaires.

Le produit VMS DECwindows/Français/CH fournit une interface en langue française
aux applications qui créent l'environnement DECwindows. Cet environnement se
compose de quatre composants de base:


- Le Gestionnaire de session représente le point d'entrée de la station de
  travail DECwindows. Il permet de lancer des applications, de personnaliser
  l'environnement général de la session, de copier et d'imprimer tout ou partie
  de l'écran, de gérer la sécurité et de contrôler la session d'une manière
  globale.


- Le Gestionnaire de fenêtres permet de paramétrer les fenêtres.


- Le Tableau de bord fournit une interface graphique à la gestion de fichiers
  VMS. Il permet de se déplacer dans l'arborescence VMS et d'effectuer des
  opérations sur les fichiers (copie, destruction, impression...).


- L'émulateur de terminal DECterm permet aux utilisateurs des stations de
  travail DECwindows de retrouver un environnement identique à celui proposé par
  les terminaux à matrices de caractères de type VT.


DECwindows comprend les applications de bureau suivantes:

- Agenda - Un utilitaire de gestion du temps.

- Bloc-notes - Éditeur de texte.

- Calculatrice - Une calculatrice scientifique.

- Courrier - L'équivalent DECwindows de l'utilitaire MAIL de VMS.

- Dessin - Éditeur graphique en mode point.

- Émulivre - Un outil de consultation de documentation à l'écran.

- Horloge - Affichage de l'heure en mode numérique ou analogique et fonction
  d'alarme.

- Pousse-pousse - Un jeu de logique.

- Répertoire - Une application de stockage hiérarchique d'informations,
  reprenant les fonctionnalités d'un annuaire. 

- Visionneuse CDA - Outil permettant d'afficher des documents composés
  (texte et graphiques conformes à l'architecture CDA) sur un terminal ou une
  station DECwindows.

Les utilisateurs de DECwindows sur stations de travail peuvent afficher les
fichiers au format PostScript (registered_symbol) grâce à la
fonction Display PostScript (registered_symbol), disponible dans
les applications Visionneuse CDA et Courrier.
       
Pour plus de détails sur VMS DECwindows, veuillez vous reporter à la
Description de Produit Logiciel du système d'exploitation VMS (SPD 25.01.XX).
