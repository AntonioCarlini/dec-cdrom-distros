          DESCRIPTION

          Le produit VMS DECwindows/Canadien-français est l'interface
          française du produit VMS DECwindows, qui est inclus dans le sys-
          tème d'exploitation VMS. Pour exécuter VMS DECwindows/Canadien-
          français, le
          système d'exploitation VMS et le produit VMS DECwindows sont
          nécessaires.

          Le produit VMS DECwindows/Canadien-français fournit une
          interface en langue française aux applications qui créent
          l'environnement DECwindows.

          Cet environnement comprend quatre éléments de base :

          o  Le Gestionnaire de session fournit une interface utilisateur
             de niveau supérieur aux postes de travail DECwindows. Il
             assure le lancement des applications, la personnalisation à
             l'échelle de la session, l'impression d'écran, la gestion de
             la sécurité et la commande des sessions.

          o  Le Gestionnaire de fenêtre donne à l'utilisateur la possibil-
             ité de gérer les fenêtres.

          o  Le Tableau de bord est une interface graphique
             destinée à la gestion des fichiers VMS et qui permet
             aux utilisateurs de se déplacer dans les fichiers VMS et
             d'effectuer des opérations sur les fichiers.

          o  L'émulateur de terminal DECterm offre aux utilisateurs de
             postes de travail une interface à cellules de caractères
             traditionnelle pour les fonctions et les applications VMS
             existantes.

          o  Un ensemble d'applications intégrées est également inclus
             dans le logiciel DECwindows. Ces applications sont les suiv-
             antes :

             o  Émulivre - Un outil de consultation de la documentation à
                l'écran.

             o  Calculatrice - Une calculatrice scientifique.

             o  Agenda - Un utilitaire de gestion du temps.

             o  Répertoire - Une application de stockage hiérarchique
                de l'information, reprenant les fonctionnalités d'un
                annuaire.

             o  Horloge - Affichage de l'heure en mode numérique ou
                analogique et avertissement à l'aide d'un bip d'alarme.

             o  Visionneuse CDA - Outil permettant d'afficher des
                documents composés (texte et graphiques conformes à
                l'architecture CDA) à un terminal ou à un poste de travail
                DECwindows.

             o  Courrier - L'équivalent DECwindows de l'utilitaire MAIL de
                VMS.

             o  Bloc-notes - Un éditeur de texte.

             o  Dessin - Un éditeur graphique en mode point.

             o  Pousse-pousse - Un jeu de logique.

          Les utilisateurs de postes de travail DECwindows peuvent af-
          ficher des fichiers Postscript[MD] garantissant une véritable
          fidélité TATI (tel affiché, tel imprimé) entre l'écran affiché
          et toutes les imprimantes PostScript de Digital. Cette possi-
          bilité d'affichage est offerte avec la visionneuse CDA et le
          courrier DECwindows.

          Pour plus de détails sur le produit VMS DECwindows, veuillez
          vous reporter à la Description du logiciel du système d'exploitation
          VMS (SPD 25.01.XX).
