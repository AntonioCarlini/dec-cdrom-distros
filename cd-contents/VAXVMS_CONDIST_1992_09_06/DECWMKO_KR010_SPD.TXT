           PRODUCT NAME: VMS DECwindows Motif/Hangul, Version 1.0   SPD 27.H4.00
                         


           DESCRIPTION

           VMS DECwindows Motif/Hangul V1.0 is the Korean version of VMS DECwin-
           dows Motif (DECwindows Motif) V1.0. When used in conjunction with VMS
           /Hangul, DECwindows Motif/Hangul V1.0 extends the standard DECwindows
           Motif V1.0 to support Korean character input/ output handling,to pro-
           vide a Korean language user interface,and to offer a Korean language
           platform for developing Korean DECwindows Motif applications.

           DECwindows Motif/Hangul supports Korean data that conforms to the KS
           C 5601-1987 standard of the Republic of Korea.

           This SPD describes only the Korean specific features of DECwindows 
           Motif/Hangul which are extensions or additions to standard VMS 
           DECwindows Motif. For a description of the base functionalities of 
           VMS DECwindows Motif, please refer to the appropriate Software 
           Product Description (SPD 36.09.00)


           FEATURES

           DECwindows Motif/Hangul provides the following Korean processing fea-
           tures in addition to the base functionalities of VMS DECwindows 
           Motif.


           Korean Input Methods

           The following input methods are supported in DECwindows Motif/Hangul
           for the input of Korean characters:

           o  Hangul

           o  Hanja

           o  Row-Column


           HangulDECterm

           HangulDECterm utility in DECwindows Motif/Hangul is the extended ver-
           sion of standard DECterm to provide terminal emulation of Korean ter-
           minals. Emulation of the following Korean terminal is currently sup-
           ported:

           o  VT382-K

           Motif Toolkit Library

           The following widgets of Motif Toolkit Library in DECwindows Motif
           /Hangul are enhanced to support Korean data processing for Korean 
           character input/output and Korean message display:

           o  BulletinBoard Widget

           o  Command Widget

           o  Label Widget

           o  List Widget

           o  MessageBox Widget

           o  RowColumn Widget

           o  Scale Widget

           o  SelectionBox Widget

           Digital Extension of Motif Toolkit Library

           The Digital's Extension of Motif Toolkit Library is also extended for
           Korean data manipulation in the following widgets with Korean char-
           acter input/output capabilities and Korean message display:

           o  ColorMix Widget

           o  CSText Widget

           o  Help Widget

           o  Print Widget

           o  Structured Visual Navigation (SVN) Widget


           X User Interface (XUI) Toolkit and UIL Compiler Support

           DECwindows Motif continues to support the XUI Toolkit for user's ex-
           isting DECwindows applications. Since Motif is an open standard user
           interface, new user applications should be written with Motif 
           Toolkit.
           

           User Environment

           The following features in DECwindows Motif have been enhanced to ac-
           cept input in Korean character and/or display of Korean data:

           o  Login Extension

           o  Session Manager

           o  FileView

           o  Motif Window Manager

           o  LinkWorks Hyperinformation Services
