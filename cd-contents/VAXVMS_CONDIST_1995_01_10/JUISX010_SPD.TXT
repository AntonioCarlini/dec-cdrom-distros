仕様書の包含する範囲

日本語VAXuisx・VWS移行ツールの概要

日本語VAXuisxランタイム・ライブラリ/VMSバージョン1.1を使用することによって，
日本語VWSウィンドウ・システム向けに作成されている既存のほとんどのアプリケー
ションをDECwindowsウィンドウ・システムの下で実行することができます。日本語
VAXuisxでは，このための描画および入力サービスのためのXlibプログラム・インター
フェイスを使用する新しいランタイム・ライブラリを提供します。UISルーチンを新し
いUISXSHRランタイム・イメージに置き換えますので，ほとんどのアプリケーションは
再リンクする必要はなく，日本語VWSをインストールする必要もありません。

機能

o  X11を使用して，ほとんどの日本語VWSルーチンをエミュレートします(DOP，共通ビ
   デオQIOインターフェイス，タブレット，および共有カラー・マップはエミュレー
   トしません)。

o  X11ルーチンを使用して，日本語VWS仮想カラー・マップをエミュレートします。

o  日本語VWSライティング・モードのすべてをサポートします。

o  標準の日本語VWSパターンは日本語VAXuisxに組み込まれます。

o  標準の英語版VWSフォントをDECwindowsフォーマットで提供します。

SIGHT(Simple Interactive Graphics Handling Tool)バージョン4.4 -このパッケージ
によってワークステーション上にグラフィックを作成することができます。これらの
グラフィックを，ハードコピー装置で出力するためにファイルに保存することができ
ます。そして，これらのファイルを更新したり，サポートされている他のVAXstation
に転送/複写することができます。

HardCopy UIS(HCUIS)バージョン4.4 -グラフィックのハードコピー出力の生成を必要
とするアプリケーションのために，HardCopy UIS(HCUIS)を提供します。HCUISによっ
て，アプリケーションでVWSディスプレイ・リストを様々なハードコピー装置で印刷す
るためのフォーマットに変換することができます。HCUISルーチンによって，アプリケ
ーションでVWSディスプレイ・リストからUISファイルを作成することもできます。ま
た，HCUISのDCLインターフェイスで，UISファイルを様々なハードコピー装置上で印刷
するためのフォーマットに変換することもできます。

UIS to DDIF Converterバージョン2.1 -このツールを使用することによって，
HardCopy UIS(これはSIGHTで作成されたファイルも含まれます)で作成されたUISメ
タファイルをCDAコンバータ・ライブラリでサポートされる任意の出力フォーマットに
変換することができます。

日本語UIS Source Code Annotatorバージョン2.0 -日本語UIS Source Code Annotator
は，日本語VWSアプリケーションをDECwindowsに移植する際に役立ちます。日本語VWS
アプリケーション・ソースに対してAnnotatorを実行すると，日本語VWSの各ルーチン
に対応した類似のXlibルーチンが示されます。また，対応するXlibルーチンがない場
合は，その旨が示されます。Annotatorは『A Guide to Migrating VWS Application
to DECwindows』に従って使用してください。

『A Guide to Migrating VWS Applications to DECwindows』バージョン1.0 -このド
キュメントは，既存の日本語VWSアプリケーションをDECwindowsプラットホームに移植
するソフトウェア開発者および設計者を支援するためのもので，2つのプラットホー
ムの相違，および移植されたアプリケーションのプログラミング例が記述されていま
す。

VWS Demo Kitバージョン4.4 - VWS Demo Kitは様々な言語で書かれたVWSアプリケー
ション集です。ほとんどのデモ・プログラムのソース・コードと実行イメージの両方
が，キットの一部分として含まれています。

ティング・システムのライセンスに含まれています。標準版VMSオペレーティング・
システムのライセンスについての詳細に関しては『VMS Operating System Software
Product Description (SPD 25.01.xx)』を参照してください。

日本語VAXuisx，および日本語UIS Source Code Annotatorを使用する権利は，日本語
VMSオペレーティング・システムのライセンスに含まれています。日本語VMSオペレー
ティング・システムのライセンスについての詳細に関しては『日本語VMSオペレーティ
