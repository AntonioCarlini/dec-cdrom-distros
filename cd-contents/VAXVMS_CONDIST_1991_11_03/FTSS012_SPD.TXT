           DESCRIPTION

           VAXft System Services is a VMS layered software product. When
           combined with base VMS and the VAXft series hardware, VAXft
           System Services provides the full fault tolerant system ca-
           pabilities. The resulting features include the following:

           o  Automatic failover

           o  Configuration management

           o  Automatic dial-out notification

           o  Component fault isolation and recovery

           o  Optional support for the DSF32-AA/AB

           VAXft System Services software is used by System Managers,
           Operations, and Digital Service personnel to perform sys-
           tem management and maintenance operations. The software is
           transparent to the system users as well as to VMS layered
           products and applications.

           HARDWARE REQUIREMENTS

           VAXft configuration as specified in the System Support Ad-
           dendum (SSA 31.64.03-x).
