            Product Name: Session Support Utility,        SPD 27.79.02
                          Version 1.3



           DESCRIPTION

           The Session Support Utility (SSU) is a VMS[TM] layered soft-
           ware product that allows users to log into and maintain mul-
           tiple independent sessions on a VMS system using Digital's
           VT330, VT340 or VT420 terminals. SSU software can be acti-
           vated on either of the two communication ports on these ter-
           minals.

           Session Support Utility supports three (3) types of connec-
           tions to a VMS host.

           o  Hardwired ASYNC line

           o  Modem Line

           o  LAT line

           HARDWARE REQUIREMENTS

           VAX, MicroVAX, VAXstation, or VAXserver configuration as spec-
           ified in the System Support Addendum (SSA 27.79.02-X).

           SOFTWARE REQUIREMENTS

           For Systems Using Terminals (No DECwindows Interface):

           VMS Operating System

           For Workstations Running VWS:

           VMS Operating System
           VMS Workstation Software

           For Workstations Running DECwindows:

           VMS Operating System (and necessary components of VMS DECwin-
           dows)
