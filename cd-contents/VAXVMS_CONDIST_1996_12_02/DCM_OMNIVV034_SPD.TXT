 DESCRIPTION

 The BASEstar Classic DAS for DEComni Software allows users to access
 devices which implement the Manufacturing Message Specification (MMS),
 ISO IS-9506, and/or the Siemens SINEC-AP and SINEC-H1 protocol through
 the BASEstar Classic Device Connect Manager (DCM).

 Via BASEstar Classic DCM, the user may configure path, type, device
 and physical point definitions, read and write data values, and con-
 figure for polled and unsolicited data collection.

 The BASEstar Classic DAS for DEComni software supports the following:

 o  Devices which implement the Manufacturing Message Specification,
    ISO IS-9506

 o  Devices which implement the SINEC-AP protocol

 o  Devices which implement the SINEC-H1 protocol

 The BASEstar Classic DAS for DEComni software supports the following
 device access functions:

 o  An Upload Domain (Programs and Data) function that transfers the
    contents of a device's memory to a VMS file

 o  A Download Domain (Programs and Data) function that transfers the
    contents of a VMS file to a device's memory

 o  Start and Stop functions (Program Invocations) that change the op-
    erating mode of a device

 o  A Read Named and Unnamed Variables function that allows users and
    applications to read data from a specific address in a device's mem-
    ory

 o  A Write Named and Unnamed Variables function that allows users and
    applications to write data to a specific address in a device's mem-
    ory

 o  A Read Device Status and Identification function

 o  A Get Device Directory function (Get Name List)

 o  A Receive Unsolicited Variables function (Information Reports, Read
    and Write Indications)

 o  Read, Write and Exchange SINEC-AP and SINEC-H1 messages (Supported
    for SINEC-AP and SINEC-H1 compliant devices only)
