     Beschreibung

     DECplan ist ein integriertes Zeit- und Projektmanagementsystem
     zur Planung, Verfolgung und Auswertung von Projekten und Bespre-
     chungen. DECplan ist eine Client/Server-Anwendung unter DECwin-
     dows und bietet einen Server und zwei Clienten, den DECplan
     Client und DECplan Time Manager Client. DECplan Time Manager
     ist Bestandteil von DECplan. Weitere Informationen über DECplan
     finden Sie in der Software-Produktbeschreibung SPD 27.L1.xx.
     Der DECplan Server verwaltet eine DECplan-Datenbank, in der die
     Planungsdaten nach verschiedenen Plantypen gespeichert werden:
     Projektpläne, persönliche Terminpläne, Pläne für Betriebsmittel
     und Pläne für Besprechungsräume. Es können mehrere DECplan oder
     DECplan Time Manager Clienten eine Verbindung zu einem einzigen
     Server aufbauen. Diese mit dem DECplan Server verbundenen Cli-
     enten ermöglichen den Benutzern, die in der DECplan-Datenbank
     gespeicherten Projektdaten gemeinsam zu nutzen, Besprechungen
     anzuberaumen und Information auszutauschen.

     Eigenschaften von DECplan Time Manager Client

     DECplan Time Manager ist Bestandteil von DECplan. Alle hier
     aufgeführten Funktionen und Eigenschaften sind auch in DECplan
     enthalten.

     Tages-, Montas- und Jahresansichten

     o  Benutzer reservieren Zeit und planen Aufgaben in ihrer Ta-
        gesanzeige des Time Managers. In DECplan werden auf dieser
        Grundlage über mehrere Pläne hinweg Zeitpläne erstellt und
        koordiniert.

     o  Benutzer können den Kalender in der Tages-, Monats- oder
        Jahresansicht anzeigen.

     o  Mit Hilfe des DECplan Servers können Kalendereinträge ver-
        schiedener Personen nach verfügbarer Zeit durchsucht werden,
        um die Zuordnung von Aufgaben und Besprechungen planen zu
        können.

     Planung von Besprechungen, Suche nach verfügbarer Zeit

     o  Benutzer können Besprechungen mit anderen Benutzern von
        DECplan oder DECplan Time Manager ansetzen, indem Sie die
        Dauer, Agenda und den Ort festlegen und eine Suche nach
        verfügbarer Zeit in den Kalendern der anderen Teilnehmer
        ausführen. Somit kann ein für alle Teilnehmer günstiger
        Termin ermittelt werden.

     o  Es können auch Verteilerlisten für Gruppen oder Netzwerk-
        Adressen verwendet werden (für Personen, die nicht mit DEC-
        plan arbeiten), um Besprechungen einzuplanen, Konferenzräume
        zu buchen und Agendas zu versenden.

     o  Besprechungen können in verschiedenen Intervallen wiederholt
        werden, z.B. täglich, alle zwei Wochen oder monatlich, oder
        aber individuell geplant werden.

     Verfolgung von Zielen

     o  Benutzer können Ziele festlegen und diesen eine Reihe von
        Aufgaben zuordnen, die zur Erreichung nötig sind. Weiterhin
        können Ziele priorisiert, der Status festgehalten und das
        Ziel mit Hilfe von DECplan Time Manager bis zur Vollendung
        verfolgt werden.

     o  Ziele können im Lauf der Zeit geändert werden, um z.B. Ände-
        rung von Prioritäten widerzuspiegeln.

     Tabellenansichten

     o  Benutzer von DECplan Time Manager können die Aufgabenli-
        ste verwenden, um Aktionspunkte und eine Liste mit offenen
        Aufgaben und Vorgängen zu vermerken. Dabei kann es sich um
        persönliche Aufgaben handeln oder um Teilaufgaben, die mit
        dem Projektmanager mit Hilfe von DECplan verhandelt und von
        diesem weitergeleitet wurden.

     o  Eigene Daten können angezeigt und bearbeitet und Tabellen-
        spalten und -zeilen angepaßt werden.

     Signal

     o  Als Erinnerung an Aufgaben und Besprechungstermine kann ein
        Signal gesetzt und hierfür in der Tagesanzeige des Kalenders
        aus einer Vielzahl von Sinnbildern das für die jeweilige
        Aktivität passende Sinnbild ausgesucht werden.

     o  Signale müssen für Besprechungen, die sich wiederholen, nur
        einmal gesetzt werden.

     o  DECplan bietet die Möglichkeit, die Zuordnung von Aufgaben
        und Besprechungen und berichteten Aufwand zu verhandeln. Da-
        bei kann eine Person Aufgaben oder Termine anderen Benutzern
        vorschlagen, diese Benutzer wiederum haben die Möglichkeit,
        die Vorschläge anzunehmen oder abzulehnen, und eine Meldung
        als Antwort auf den Vorschlag zu schicken. Sobald ein Vor-
        schlag akzeptiert wurde, erscheint im Plan der angesprochenen
        Person ein entsprechender Vermerk.

     o  Es besteht auch die Möglichkeit, einen Vorschlag als ver-
        pflichtend zu vermerken. Dies ist z.B. sinnvoll, wenn eine
        Verhandlung bestimmter Punkte nicht nötig oder erwünscht ist.

     Berichteter Aufwand

     o  Benutzer können geleistete Arbeit in der Tagesanzeige oder
        in einer tabellarischen Ansicht berichten. Daten aus be-
        richtetem Aufwand werden auf den tatsächlichen Aufwand einer

        Aufgabe aufgerechnet und somit der Fortschritt einer Auf-
        gabe angezeigt. Wenn der Status auf diese Weise mit Hilfe
        von Time Manager aktualisiert wird, werden automatisch die
        Projektmanagement-Ansichten von DECplan aktualisiert.

     o  Berichteter Aufwand kann entweder von Benutzern verhandelt
        werden oder aber vom DECplan Server automatisch in bestimmten
        Abständen eingesammelt und verbucht werden.

     Schlüsselwörter, Sortierfunktion, Filter

     o  In DECplan kann der Benutzer einer Aufgabe oder anderen Ob-
        jekten in der Datenbank beliebige alphanumerische Zeichen-
        folgen zuordnen. Diese Schlüsselwörter können zum Sortieren
        oder Filtern von Informationen in Ansichten und Berichten
        verwendet werden.

     o  Für alle tabellarischen Ansichten gibt es eine einheitliche
        Sortierfunktion, die auf- und absteigende Sortierung und
        Sortieren in Gruppen ermöglicht.

     o  Die Filter-Funktionalität ermöglicht es dem Benutzer, be-
        liebige Informationen aus der Datenbank zu ziehen, die in
        Berichten, Diagrammen und Vorgangslisten verwendet werden.

     o  Der Benutzer kann mehrstufige Filteranweisungen erstellen,
        um bestimmte Daten aus einem oder mehreren Projektplänen
        auszuwählen und anzuzeigen.

     Exportieren/Importieren

     o  DECplan bietet die Möglichkeit, mit Hilfe des DTIF-CDA-
        Austauschformats Daten zu importieren. Es gibt verschiedene
        erweiterte CDA-Konvertierprogramme, mit denen Daten von einem
        anderen Format in das DTIF-Format konvertiert werden können.
        Daten im DITF-Format können dann in die DECplan-Datenbank
        importiert werden.

     o  Es können vollständige DECplan-Pläne im DTIF-Format aus
        DECplan exportiert und in andere Anwendungen importiert
        werden, die dieses Format unterstützen, wie z.B. DECdecision.
        Der exportierte Plan kann über Server wieder in DECplan
        zurück importiert werden.

     o  Aus DECplan können grafische und tabellarische Ansichten
        und Berichte im DDIF-Format exportiert werden. Dieses Format
        wird von anderen DECwindows-Anwendungen, wie z.B. DECwrite,
        Elektronische Post von DECwindows und DECpresent unterstützt.

     Online-Hilfe

     o  Zusätzlich zu den oben erwähnten Funktionen und Eigenschaften
        bietet DECplan kontext-sensitive Online-Hilfe zur Unterstüt-
        zung bei der Arbeit mit DECplan und Time Manager.

     Konvertierprogramme

     o  Benutzer des Kalenders von DECwindows können ihre eigenen
        Kalender-Datenbanken in DECplan-Datenbanken konvertieren.
        Dieses Konvertierprogramm ist in DECplan enthalten.
